iOS App (Capacitor)

To convert this into a native iOS App:

Run npm init and install Capacitor (npm i @capacitor/core @capacitor/cli @capacitor/ios).

Initialize: npx cap init.

Add iOS platform: npx cap add ios.

Open Xcode: npx cap open ios.

Important: Add Privacy - Camera Usage Description to your Info.plist to prevent crashes when taking photos.
